Firebase — Conexión Online

Conecta tu base de datos para que el panel y la web funcionen en línea desde cualquier lugar.

Estado de Firebase
Sin configurar — usando almacenamiento local
Una vez configurado, el Panel escribe en Firestore y la Web lee en tiempo real. Funciona desde cualquier link o dispositivo.
Credenciales del Proyecto Firebase
Cómo configurar Firebase (gratis)
1Crea un proyecto

Ve a console.firebase.google.com → Agregar proyecto → sigue los pasos (gratis).

2Agrega una app web

En tu proyecto → icono </> → Registra la app → Copia el objeto firebaseConfig.

3Habilita Firestore

En el menú lateral → Firestore Database → Crear base de datos → Modo producción → elige región.

4Configura las reglas de seguridad

En Firestore → pestaña Reglas → Pega este código para permitir lectura pública y escritura solo autenticada (o usa el modo desarrollo):

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /catalog/{doc} {
      allow read: if true;
      allow write: if true; // Cambia a auth en producción
    }
    match /views/{doc} {
      allow read, write: if true;
    }
  }
}
5Pega las credenciales arriba y dale Guardar

El panel se conectará automáticamente y sincronizará los datos con la web.
